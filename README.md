# 🌿 Plant Identification Online

Plant Identification Online is an AI-powered web application that identifies plants from uploaded images using the Pl@ntNet API and provides trusted botanical information from Wikipedia.

---

## ✨ Features

- 🌱 AI-powered plant identification
- 📖 Plant descriptions
- 📜 History & Region
- ❤️ Benefits & Uses
- 📊 Confidence score
- ❌ Plant Not Found detection
- 📱 Mobile-friendly design
- ⚡ Fast and easy to use

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Pl@ntNet API
- Wikipedia API

---

## 📷 How to Use

1. Open the website.
2. Upload a plant image.
3. Click **Identify Plant**.
4. View the plant's:
   - Common Name
   - Scientific Name
   - Family
   - Confidence Score
   - Description
   - History & Region
   - Benefits & Uses
